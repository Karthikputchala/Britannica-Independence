
# Britannica Independence

Project is based on the concept of text similarity between any two documents.


## Introduction

There are many people who are known to fight for the Independece of their country.From many of them BRITANNICA (Encyclop√¶dia publishing company) had made a document database of most prominently known Freedom fighters. In this project we had collected the data from [BRITANNICA](https://www.britannica.com/biographies/history/independence) and had done the similarity comparision between any two documents as the User selected.
